## About

`@nanoforge-dev/component-system` is a package containing a collection of generical component and system to create games.

## Installation

**Node.js 25 or newer is required.**

```sh
npm install @nanoforge-dev/component-system
yarn add @nanoforge-dev/component-system
pnpm add @nanoforge-dev/component-system
bun add @nanoforge-dev/component-system
```
